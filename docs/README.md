# ChipLens Documentation

Welcome to the official engineering documentation for **ChipLens**.

ChipLens is a modular RTL analysis, verification, and repair framework designed to unify simulation, formal verification, diagnostics, coverage analysis, and AI-assisted design understanding.

## Philosophy

ChipLens is designed as a **verification platform**, not merely a wrapper around existing EDA tools.

Every subsystem should be:

- Modular
- Testable
- Extensible
- Well documented
- Backend independent where practical

The documentation evolves together with the source code.
