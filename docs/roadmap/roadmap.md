# ChipLens Roadmap

## Vision

ChipLens aims to become a professional RTL analysis and verification platform that combines:

- RTL parsing
- Static analysis
- Simulation
- Coverage analysis
- Formal verification
- AI-assisted repair
- Design understanding
- Research-grade verification workflows

The long-term goal is to provide a modular verification framework rather than a frontend for individual EDA tools.
